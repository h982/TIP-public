## :speech_balloon: 프로젝트 소개

- 서비스명 : Tip
- 소재 : To do list (PJT 관리) 시스템 동시편집 기능 개발
- 하나의 프로젝트 안에 할 일(Todo)를 여러 이해 관계자들이 동시 편집, 관리하는 시스템
  - 프로젝트 안 팀 별 할 일을 한 눈에 파악할 수 있다.
  - 할 일이 부서 간 이동할 수 있다.
  - 할 일에 연관된 세부 설명, URL을 저장할 수 있다.
